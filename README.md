# PeluxDeviceManager
Pelux Suite Device Manager

This library provides a unified API to query and interact with all kinds of hardware devices. 

Current backends:
- Solid

## Dependencies
- Solid (http://solid.kde.org) + deps: libupnp, flex/bison, udev
